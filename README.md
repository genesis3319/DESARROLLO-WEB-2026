# Arte Mostacilla - Bisutería Artesanal

## Descripción

Arte Mostacilla es un sitio web desarrollado como parte del Proyecto Integrador de la asignatura Desarrollo de Aplicaciones Web de la Universidad Estatal Amazónica.

La página presenta información sobre un emprendimiento de bisutería artesanal elaborado con mostacillas, incluyendo una sección de inicio, información sobre el emprendimiento, productos, contacto, promociones y un formulario para comunicación con los clientes.

## Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap 5

## Contenido del sitio

* Inicio
* Quiénes Somos
* Productos
* Contacto
* Video de referencia
* Promociones y horarios
